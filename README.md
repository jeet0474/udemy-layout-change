# Udemy Layout Change

A lightweight browser extension to customize and improve the Udemy course layout.

This project aims to make Udemy more usable, flexible, and closer to the classic experience by allowing dynamic layout adjustments (e.g., resizing panels, improving spacing, enhancing navigation).

---

## ✨ Features

* Resize and adjust lecture layout dynamically
* Improve vertical space usage
* Persistent layout (remembers your settings)
* Works directly inside Udemy pages
* Built for extensibility (more layout tools coming)

---

## 🚀 Getting Started

### 1. Download the extension

* Click **Code → Download ZIP**
* Or clone the repository:

```bash
git clone https://github.com/<your-username>/udemy-layout-change.git
```

---

### 2. Extract the files

If you downloaded ZIP:

* Right-click → **Extract All**
* Open the extracted folder

---

### 3. Load extension in Chrome

1. Open:

```
chrome://extensions/
```

2. Enable **Developer Mode** (top right)

3. Click **Load unpacked**

4. Select the extracted project folder

---

### 4. Use the extension

* Open any Udemy course lecture page
* The layout tools will automatically activate
* Interact with UI elements (like draggable edges) to adjust layout

---

## 🧩 How It Works

* Injects custom scripts into Udemy pages
* Detects layout sections (video, notes, panels)
* Adds interactive controls (e.g., drag to resize)
* Applies dynamic CSS changes
* Saves preferences using browser storage

---

## ⚙️ Tech Stack

* JavaScript (Vanilla)
* Chrome Extension (Manifest v3)
* DOM manipulation + CSS overrides

---

## 📌 Notes

* Works only on `udemy.com`
* Layout may break if Udemy updates their UI structure
* Reload the extension if changes stop working
